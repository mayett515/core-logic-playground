# Migration from v1

v1 was organized as a helpful knowledge pack.

v2 is organized as a routed execution system.

Main changes:

```text
JSON routing summaries    → optional only
Markdown knowledge files  → YAML + XML rule contracts
Folders for reference     → flat `.ai-rules` control plane
Concept docs              → learner-state routing
Checklist file            → anti-regression contract
```

The learning philosophy is the same. The obedience mechanics are stronger.
