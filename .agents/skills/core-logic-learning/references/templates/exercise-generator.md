# Exercise Generator Template

Generate exercises that test derivation.

```text
For [CONCEPT], create:
1. One recognition exercise: identify the underlying pattern.
2. One derivation exercise: reconstruct syntax from the concept.
3. One transfer exercise: map the idea to another language/tool.
4. One fragility exercise: find where a shallow analogy fails.
```

Avoid exercises that only ask the learner to recall syntax.
