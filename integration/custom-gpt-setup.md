# Custom GPT / Gem Setup

Use this setup:

```text
Instructions field:
→ paste GPT_INSTRUCTIONS_COPYPASTE.md

Knowledge uploads:
→ upload the full unzipped folder
```

The instructions field contains the behavioral contract.

The uploaded files contain the routing system, examples, prompt cards, templates, and fallback docs.

Do not paste raw links into instructions. Links are weaker than directly uploaded knowledge.
