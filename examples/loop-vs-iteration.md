# Example: Loop vs Iteration

Beginner model:

> A loop is `for (...) { ... }`.

Core Logic model:

> A loop is one syntax for iteration: repeated action over time, count, or data.

Transfer map:

```text
for loop        → manual iteration
while loop      → condition-controlled iteration
map             → transform each item
filter          → keep items matching a condition
database query  → select/filter/project rows
stream pipeline → process items over time
AI generation   → repeated next-token prediction
```

Different syntax. Same pattern.
