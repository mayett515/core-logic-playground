# New Concept Worksheet

Use this for any new concept.

```text
Concept:

1. What problem existed before this concept?
2. What is the smallest stable mental model?
3. What does the machine/runtime do?
4. What information flows through it?
5. What old concept is this really?
6. What syntax does this language use for it?
7. Where does the analogy break?
8. What harder example would break a shallow understanding?
```
