# Cheat Sheet Addendum: Learning Coach

Add this to `CHEATSHEET.md`.

## Quiz Me / One-Liner Questions

Use this when you do not want a coding exercise.

```text
$learning-coach

Quiz me on <topic>.
Ask one question at a time.
Do not give the answer before I try.
```

## Check My Understanding

```text
$learning-coach

Check whether I understood <topic>.
Ask 3 one-liner questions.
One question at a time.
```

## Find My Weak Spot

```text
$learning-coach

Find my weak spot in <topic>.
Ask one question at a time.
If I answer badly, tell me what assumption is missing.
```

## Full Learning Loop

```text
$core-logic-learning

Teach me <topic> from first principles.

$learning-coach

Ask me 3 one-liner questions.

$core-logic-learning

Explain the weak spots.

$enterprise-typescript-research

Show me how production repos use it.
```
