# Cheat Sheet Addendum: Mermaid

## Add a diagram

```text
$mermaid-diagram-designer

Visualize <topic>.
Choose the best Mermaid diagram type.
Keep it small.
```

## Add diagram to a note

```text
$mermaid-diagram-designer
$learning-note-writer

Add a Mermaid diagram to notes/concepts/<topic>.md.
Explain how to read it.
```

## When Mermaid helps

Use it for:

- information flow,
- state changes,
- async / promises,
- request / response,
- module boundaries,
- dependencies,
- architecture,
- loops / control flow.

Do not use Mermaid if text or code is clearer.
